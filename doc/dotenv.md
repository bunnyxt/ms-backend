# .env配置项说明

| 配置项 | 说明 | 填写示例 |
| --- | --- | --- |
| `PORT` | 本后端API程序启动的端口号 | `5222` |
| `MYSQL_CONNECTION_LIMIT` | `MySQL`连接池最大连接数 | `100` |
| `MYSQL_USER` | `MySQL`连接用户名 | `username` |
| `MYSQL_HOST` | `MySQL`连接地址 | `127.0.0.1` |
| `MYSQL_PASSWORD` | `MySQL`连接密码 | `passw0rd` |
| `MYSQL_DATABASE` | `MySQL`连接默认数据库名 | `tombus` |
